Sobre a LUMINA Tech

Inovação e dados para o futuro da mobilidade.

Este sistema foi desenvolvido como parte de uma iniciativa educacional em parceria com a Ford e o SENAI CIMATEC. Nosso objetivo é fornecer uma interface intuitiva para visualização e gerenciamento de dados telemétricos de veículos.

Através do nosso Dashboard, administradores podem monitorar o status de conectividade, atualizações de software e volume de vendas de modelos como Ranger, Mustang e Territory.

+ Conectividade

Integração em tempo real com a API de dados dos veículos.

+ Segurança

Acesso restrito e controle de usuários autenticados.
